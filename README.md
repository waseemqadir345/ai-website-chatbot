# 🤖 AI Customer Support Chatbot

A custom **AI chatbot for business websites**, powered by the **ChatGPT API**.
It answers customer questions 24/7 using your own business info, and embeds into any website easily.

> Helped a US service business handle common customer questions automatically — reducing support emails significantly.

---

## ✨ Features

- ✅ Powered by OpenAI's ChatGPT API
- ✅ Trained on your own business info (services, pricing, FAQs)
- ✅ Clean, modern chat widget (bottom-right corner)
- ✅ 24/7 instant, human-like replies
- ✅ Easy to customize and deploy
- ✅ Mobile-friendly design

---

## 🛠️ Tech Stack

- **Python 3** + **Flask** – backend server
- **OpenAI API** – AI responses
- **HTML, CSS, JavaScript** – chat widget
- **python-dotenv** – secure API key handling

---

## 🚀 Installation

```bash
# 1. Clone the repo
git clone https://github.com/waseemqadir345/ai-website-chatbot.git
cd ai-website-chatbot

# 2. Install dependencies
pip install -r requirements.txt

# 3. Add your OpenAI API key
#    Copy .env.example to .env and paste your key inside
```

📖 Usage

python app.py


Then open your browser at:

http://localhost:5000


Click the 💬 button in the corner and start chatting!


⚙️ Customize for Your Business
Open app.py and edit the BUSINESS_INFO section with your own:

Services
Pricing
Hours
Contact details

The chatbot will answer based on that information.
